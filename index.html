MTA Styles

The colors of the MTA in a easy to include CSS/SASS/LESS/JSON library.

Include

You can easily install MTA Styles with bower install mta-styles or npm install mta-styles.

There is also LESS and SASS versions with relevant variables for each line.

Or just use the regular mta-styles.css CSS file.

Symbols

It's easy to put a MTA line symbol in your markup, just use some markup like <i class="mta mta-symbol subway-n"><i> suffix. Some examples:

The symbols will adjust to the font-size they are placed in, but you can also use classes to make the symbols small, medium, and large:

mta-small mta-med mta-large

Backgrounds

MTA Styles provides all the colors as background with CSS selectors like .mta .mta-subway-a. Some examples:

Subway A line
mta subway-a
Long Island Rail Road Babylon branch
mta lirr-babylon
Metro-North Railroad Pascack Valley line
mta metro-north-pascack-valley

Foregrounds

You can also use the line colors as foreground colors by adding a -fg suffix. Some examples:

Subway J line
mta subway-j-fg
Long Island Rail Road Montauk branch
mta lirr-montauk-fg
Metro-North Railroad Harlem line
metro-north-harlem-fg

JSON

MTA Styles also provides a JSON file for use in applications or data analysis.
